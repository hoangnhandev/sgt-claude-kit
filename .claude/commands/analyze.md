# Analyze Workflow

## 1. Requirement Analysis

**Agent**: `requirement-analyst`
**Input**: Feature request / Issue.
**Task**: Parse requirements into structured specs.

## 2. Codebase Scouting

**Agent**: `codebase-scout`
**Input**: Analyzed Requirements.
**Task**: Explore codebase, identify dependencies and impact areas.

## Output

- Memory: Requirements & Codebase Analysis entities.
- Ready for: `/plan`
