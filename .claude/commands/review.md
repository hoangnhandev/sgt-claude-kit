# Review Workflow

## 1. Review

**Agent**: `code-reviewer`
**Input**: Git diff.
**Task**: Check Security, Logic, Style.

## 2. Output

**Status**: `Approved` or `Changes Requested`.
**Artifact**: Review Report.
