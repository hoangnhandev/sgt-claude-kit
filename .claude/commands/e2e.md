# E2E Testing Workflow

## 1. Setup

**Agent**: `test-engineer` (Mode: E2E)
**Task**: Check env, launch browser.

## 2. Execute

**Agent**: `test-engineer`
**Tools**: Use Playwright MCP (`navigate`, `click`, `screenshot`).
**Task**: Run scenario steps.

## 3. Report

**Output**: Pass/Fail with screenshots.
